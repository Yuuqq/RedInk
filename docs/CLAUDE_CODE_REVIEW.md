# Claude Code 互审用 Prompt（可选）

你可以把下面的 prompt 直接丢给 Claude Code，让它对 RedInk 做一次结构化 code review，然后把它的建议贴回来，我可以再帮你落地。

```text
你是资深全栈工程师，请对这个仓库做一次 code review：

目标：
1) 作为本地可用的图文生成工具（Flask + Vue）运行稳定
2) 兼容 OpenAI-compatible 代理（例如本地 CLIProxyAPI）
3) 关注并发/内存/错误处理/可观测性/安全性

请按以下结构输出：
- 高优先级问题（必须修）
- 中优先级优化（建议修）
- 低优先级改进（可选）
- 你建议新增/调整的 API（包含路径/请求/响应草案）
- 你建议新增的测试用例清单

限制：不要泛泛而谈，尽量引用具体文件路径和函数名。
```
